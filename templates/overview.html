in this application i create many endpoint.
for get request i used URL.
and for post request i used postmane.

these are my application endpoint given below 


1) /api/get_name/<name>

method : GET

INPUT - book name or a term in the name of the book
OUTPUT - list of all the books with that name or term in the book name 

2) /api/get_rent/<minR>/<maxR>

method : GET

INPUT - rent price range
OUTPUT - list of books with rent in that range

3) /api/get_list/<name>/<cat>/<rent>

method : GET

INPUT - category + name/term + rent per day(range)
OUTPUT - list of books with matching values as in input 

4) /api/book_issue

method : POST

INPUT - book name + person name + issue date (BOOK IS ISSUED)
OUTPUT - update the Transaction DB with these values 

5) /api/book_return

method : POST

INPUT - book name + person name + return date (BOOK IS RETURNED)
OUTPUT - calculate rent based on issue date and return date and update the Transaction DB with these values 

6) /api/people_list<name>

method : GET

INPUT - Book name
OUTPUT - List of people who have issued that book ( total count) + list of people who currently have that book issued ( currently issued only). 

7) /api/total_rent<name>

method : POST

INPUT - Book name
OUTPUT - Total rent generated by that book 

8) /api/booklist_peron<name>

method : GET

INPUT - Person’s name
OUTPUT - List of books issued to that person 

9) /api/date_range/<mindate>/<maxdate>

method : GET

INPUT - Date range
OUTPUT - list of books issued in that date range and the person they are issued to
